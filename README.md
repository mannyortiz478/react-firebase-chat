### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/mannyortiz478/react-firebase-chat.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Enter your firebase config in `firebase-config.js` or just use the default one
5. Start the app with a local address
    ```sh
    npm start
    ```
